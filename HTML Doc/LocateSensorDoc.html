Locate and Navigate to Sensors

This documentation outlines the implementation for the Issue that allows users to locate and navigate to sensors.

Overview

The purpose of this feature is to help users locate sensors and navigate to them. Each sensor has associated latitude and longitude values that are displayed to the user when they click the appopriate sensor in the view

Key Components

  • LoginPage: Login page for the user to login
  • HomePage: This page contains a button labeled "Navigate and Locate Sensors". When pressed, it takes the user to page with the sensor information.
  • AppData: Holds a shared list of sensor objects, each containing name, latitude, longitude, and location data.

User Flow

  • The user opens the app to the Login page and logs in.
  • The user then selects the Navigate and locate sensor option.
  • Which then displays the three sensors fo the user to see
  • Then clicking on a sensor reveals information regarding the sensor, its location etc.
